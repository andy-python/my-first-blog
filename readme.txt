## Activate the local environment (on unix)

- open terminal, go to djangogirls
- activate development environment
  source myvenv/bin/activate
  you notice (myenv) in the terminal
- Start the webserver
  source myvenv/bin/activate
  You can see it running now on 
    http://127.0.0.1:8000/ 
    http://127.0.0.1:8000/covid
  in your browser.
